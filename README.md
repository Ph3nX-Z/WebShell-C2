# WebShell C2
Web Based C2

VERSION : v1.0

![](./readme_img/home.png)


## Presentation

Webshell C2 is a c2 that use only GET requests to gain access on a device.

## Usage

### Linux/Windows :

#### Install
```sh
python3 server.py
```
or
```sh
docker build ./
docker run -it <name>
```
#### Uninstall
```sh
rm -rf WebShell-C2/
```
```sh
docker system prune -a
```
